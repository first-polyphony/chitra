# Code of Conduct

## Our Pledge

We want participation in chitra — issues, PRs, discussions — to be a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Expected Behavior

- Be respectful and considerate in comments, reviews, and issue discussion.
- Give and accept constructive feedback gracefully.
- Focus criticism on the code and the idea, not the person.
- Assume good faith from other participants.

## Unacceptable Behavior

- Harassment, insults, or personal attacks, public or private.
- Publishing others' private information without consent.
- Any conduct that would reasonably be considered inappropriate in a professional setting.

## Scope

This applies to all project spaces (issues, PRs, discussions, and any other official chitra communication channels).

## Enforcement

This is a small, best-effort maintained project — there is no dedicated enforcement team and no SLA. Report violations the same way you'd report a security issue: via [GitHub's private vulnerability reporting](https://github.com/ReticleWorks/chitra/security/advisories/new), which reaches the maintainers privately. We'll acknowledge reports within a week where possible.

This document is a short-form adaptation of the [Contributor Covenant](https://www.contributor-covenant.org/).
